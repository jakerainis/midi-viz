# Drum MIDI Preview Tool

This project is a web-based tool built with React, TypeScript, and Vite for previewing drum-based MIDI files. It allows users to:

- Upload a folder of .midi files and drum samples
- Visualize drum MIDI events on a timeline
- Control playback tempo
- Preview MIDI files using mapped drum samples

## Tech Stack

- React + TypeScript + Vite
- [@tonejs/midi](https://github.com/Tonejs/Midi) for MIDI parsing
- [Tone.js](https://tonejs.github.io/) for audio scheduling and playback

## Getting Started

1. Install dependencies:

   ```sh
   npm install
   ```

2. Start the development server:

   ```sh
   npm run dev
   ```

## Features (Planned)

- Upload and parse MIDI files
- Upload and map drum samples
